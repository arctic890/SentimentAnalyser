In this model, nltk and wordnet, one of the nltk corpus reader, are used for feature extraction.
Therefore the installment of nltk library and wordnet are required.
The wordnet data can be downloaded by executing NLTK Downloader through python command.
Manual installment in github ia also possible.
seaborn and matplotlib are the tools to display the confusion matrix.
